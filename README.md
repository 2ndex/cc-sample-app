# はじめてのClaude Code Git

Claude Code を使った Git 操作の学習用リポジトリです。

## アプリの概要

Claude Code（Anthropic の公式 CLI）を活用して、Git の基本操作を体験するためのサンプルプロジェクトです。
リポジトリの初期化からブランチ管理、コミットまでの一連のワークフローを学ぶことができます。

## 主な機能

- Git リポジトリの初期化
- ブランチの作成・切り替え
- ファイルのステージングとコミット
- Claude Code を通じた自然言語による Git 操作

## 起動方法

### 前提条件

- Git がインストールされていること
- Claude Code がインストールされていること

### セットアップ

```bash
# リポジトリのクローン
git clone <リポジトリURL>
cd git-sample

# Claude Code の起動
claude
```
